numerical_integration¶

numerical_integration.chebgauss(n)¶

Computes abscissas and weights for the Gauss-Chebyshev quadrature of second kind.

numerical_integration.chebgauss_integrate(f, eps=1e-10, max_iter=10)¶

Computes the integral of f(x) by using Gauss-Chebyshev quadrature of second kind.

References

Perez-Jorda, J., San-Fabian, E. & Moscardo, F. A simple, reliable and efficient scheme for automatic numerical integration. Comput. Phys. Commun. 70, 271-284 (1992).

Parameters:
  • f (function, auto-callable) – Function \(f(x)\), to be integrate.
  • eps (float64, optional) – Tolerance, Default is \(10^{-10}\)
  • max_iter (int, optional) – Maximum number of iterations. Default is \(10\)
Returns:

Integral value and error (out, err).
Return type:

tuple
numerical_integration.chebgauss_rq(n, rescale=False)¶

Computes the abscissas and weights for transformed Gauss-Chebyshev quadrature of second kind.

Special case for the integral \(\int f(x) dx\). since Gauss-Chebyshev is for \(\int f(x) \sqrt{1-x^2}\).

References

Perez-Jorda, J., San-Fabian, E. & Moscardo, F. A simple, reliable and efficient scheme for automatic numerical integration. Comput. Phys. Commun. 70, 271-284 (1992).

Parameters:rescale (bool, optional) – Whether the standard interval \(-1<x<+1\) has to be rescaled to the semi-infinite interval \(1<x<\infty\). Default is False.
Returns:Array with the abscissas and weights(r, w).
Return type:tuple
numerical_integration.lebedev_q(n)¶

Computes the Lebedev points and weights for spherical integration.

References

V.I. Lebedev, and D.N. Laikov, Doklady Mathematics, 59, No. 3, 477 (1999)

Parameters:n (int) – Number of angular points.
Returns:lq – Array with the coordinates and weights ([:,0] phi, [:,1] theta, [:,2] w).
Return type:numpy.ndarray
Raises:ValueError – If n is not supported.
