composite_particle¶

class composite_particle.CompositeParticle(name='none')¶

Defines a composite particle, i.e. atoms, molecules or nuclei. Since a composite particle can be a molecule, it can be added different atoms and elementary particles to the molecule. The key for elementary particles are its symbols and for atoms the key is atoms.

Parameters:name (str) – Name of the object.
add_atom(symbol, position, BOA=True, mass_number=0, units='Angstroms')¶

Adds an atom to the composite particle.

Parameters:
  • symbol (str) – Symbol of the atom.
  • position (float64[3]) – Origin of the atom (cartesian coordinates)
  • BOA (bool, optional) – Whether the atom nuclei will be treated in the BOA approach or not. Default is True
  • mass_number (int, optional) – Mass number of element ‘symbol’ \(:= A = p^+ + n^o\). If 0 the system will choose the most abundant one. Default is 0.
  • units (str, optional) – Units of the origin, valid values are ‘Angstroms’ or ‘Bohr’
add_elementary_particle(symbol, position, size=1, units='Angstroms')¶

Adds an elementary particle into the composite particle.

Parameters:
  • symbol (str) – Symbol of the elementary particle.
  • position (float64[3]) – Origin of the elementary particle (cartesian coordinates)
  • size (int) – Number of elementary particle to be added.
  • units (str, optional) – Units of the origin, valid values are ‘Angstroms’ or ‘Bohr’

size(key='all')¶

Calculates the number of kinds of particles composing the object. A ‘kind of particles’ can be atoms, e-, etc.

Parameters:key (str, optional) – Can be atoms or the key (symbol) of a given elementary particle. Default is 'all'
Returns:If all is given The size will be the number of kinds of particles. If key is given, size will be the size o the stack of a given kind of particles. For instance, in a CompositeParticle composed by atoms:{O, O}, e-:{e-(O), e-(O), e-} the key all will return 2 e- and atoms,but if key e- is given the result will be 3. note that is NOT the number of particles, because each O atom has 8 e-, in total there are 17 e-. To calculate that number use CompositeParticle.n_particles.
Return type:int
Raises:KeyError – If key is not present in the object.
