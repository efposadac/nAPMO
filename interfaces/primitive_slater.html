primitive_slater¶

class primitive_slater.PrimitiveSlater(exponent, coefficient=1.0, n=1, l=0, m=0, origin=array([ 0., 0., 0.]))¶

Slater-Type orbitals. (dict):

The slater orbital function is:

\(\chi_{p \lambda \alpha}(r, \theta, \phi) = R_{ \lambda p}(r) Y_{ \lambda \alpha}( \theta, \phi)\)

Where,

\(R_{\lambda p} = [(2n_{\lambda p})!]^{-1/2}(2\zeta_{\lambda p})^{n_{\lambda p}+1/2} r^{\lambda p -1} e^{-\zeta_{\lambda p} r}\)

And,

\(Y_{\lambda \alpha}(\theta,\phi)\) is an spherical harmonic.

References

    1. Clementi, C. Roetti, Roothaan-Hartree-Fock atomic wavefunctions. At. Data Nucl. Data Tables. 14. (3)-(4), 177-478 (1974).
Parameters:
  • zeta (float64) – Slater exponent.
  • n (int) – Quantum number n = 1, 2, 3, etc.
  • l (int) – Quantum number l = S=0, P=1, D=2, etc.
  • m (int) – Quantum number m = order of the harmonic, m <= l
  • origin (numpy.ndarray(3)) – coordinates (spherical)
