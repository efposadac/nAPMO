primitive_gaussian¶

class primitive_gaussian.PrimitiveGaussian(exponent=0.0, coefficient=1.0, l=array([0, 0, 0]), origin=array([ 0., 0., 0.]))¶

Defines a Cartesian primitive Gaussian type orbital (GTO).

compute(coord)¶

Computes the value of the object at coord.

Returns the coefficient.

get_exponent()¶

Returns the angular moment index l[3].

Returns the normalization constant.

Returns the coordinates of the object r[3].
